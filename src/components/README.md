VISAVA SHWAN ASHRAM & DOG HOSTEL

A modern and responsive animal welfare website built using React + Vite + Tailwind CSS + Framer Motion + Supabase.
The website helps showcase the work of VISAVA SHWAN ASHRAM, accept donations, display donor lists, and spread awareness for stray dog welfare.

🌟 Features
Modern responsive UI
Animated sections using Framer Motion
Donation system with QR payment upload
Supabase database integration
Live donor list
Team section
Impact statistics section
Google Maps location integration
Animated footer gallery
Mobile friendly design
Fast Vite performance
🛠️ Tech Stack
React.js
Vite

🚀 Installation

Clone the repository:

git clone https://github.com/YOUR_USERNAME/visava.git

Go inside project folder:

cd visava

Install dependencies:

npm install

Run development server:

npm run dev
🔥 Supabase Setup

Create a .env file:

VITE_SUPABASE_URL=YOUR_SUPABASE_URL
VITE_SUPABASE_ANON_KEY=YOUR_SUPABASE_KEY
🌍 Deployment

The project is deployed using Vercel.

To deploy:

Push project to GitHub
Import repository in Vercel
Deploy
